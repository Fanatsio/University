PDF — английская аббревиатура , которая может означать: Portable Document Format — формат электронных документов. PDF/A — стандарт ISO 19005-1:2005 для долгосрочного архивного хранения электронных документов. PDF/X — стандартный формат файлов, специально предназначенный для обмена готовыми к печати документами. PDF.js — свободное программное обеспечение на HTML5 и JavaScript для преобразования файлов в формате PDF в код HTML5. Portable Data File (PDF417) — двумерный штрихкод. Pop Directional Formatting — управляющий символ (Unicode U+202C) Примечания Список значений, для которых используется одна и та же аббревиатура.
